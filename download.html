About J2EE

J2EE Introduction

  • BACKGROUND OF JAVA

Java is now the acceptable and mature technology most commonly behind the strategic plans for the enterprise. Into an big corporate world using Java as a language of choice over C and COBOL for most of their new development functions like database repair and warehousing and data capture from external data feeds.

Java’s appeal lies not only in its affinity for the network and distributed computing but in Java’s other qualities such as ease of programming and cross platform capabilities -  the Write once, Run Anywhere. 

A large portion of the appeal of Java is the ease it allows in creating Web-based, self service applications that enable customers to do the work and perform other tasks over the Internet through a browser. 

J2EE application servers provide infrastructure to execute and maintain business critical applications, specially e-business critical applications. 

J2EE defines the programming model and provides underlying services and developers quickly build networked Java applications and effectively deploy them in the distributed world.
